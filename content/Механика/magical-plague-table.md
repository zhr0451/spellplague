# Степени магического безумия

| Степень | Дары                                                                               | Негативные последствия                                     |
| ------- | ---------------------------------------------------------------------------------- | ---------------------------------------------------------- |
| 1       | -                                                                                  | Помеха при проверках характеристик (Инт, Муд, Хар)         |
| 2       | Тёмное зрение 60 фт (если было, то +10 фт)                                         | Длительный отдых +2 часа                                   |
